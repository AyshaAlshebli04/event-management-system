# Event Management System

A full-stack web application developed as a **university group project** for managing events and schedules through a user-friendly web interface. The application was built using **Java Enterprise technologies** following the **Model-View-Controller (MVC)** architecture to provide a modular and maintainable codebase.

## Technical Architecture

The application follows a layered architecture that separates business logic, data access, and presentation to improve maintainability and scalability.

### Technology Stack

- **Architecture Pattern:** Model-View-Controller (MVC)
- **Backend:** Java Servlets, JSP (JavaServer Pages)
- **Database Connectivity:** JDBC
- **Database:** MySQL
- **Web Server:** GlassFish Server
- **Frontend:** HTML, CSS, JavaScript

## Project Structure

```text
Event_Management_System/
├── src/java/com/eventapp/
│   ├── dao/          # Data Access Objects (EventDAO, UserDAO)
│   ├── model/        # Domain Models (User, Event)
│   ├── util/         # Database utilities (DBUtil)
│   └── web/          # Servlet Controllers
└── web/
    ├── css/          # Stylesheets
    ├── js/           # JavaScript files
    └── *.jsp         # JSP pages
```
